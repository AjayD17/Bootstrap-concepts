1️.) Properties:

    m → margin
    p → padding

2️.) Sides:

    t → top
    b → bottom
    s → start (left in LTR)
    e → end (right in LTR)
    x → left + right
    y → top + bottom
    blank → all sides

3️.) Size Scale:

    Bootstrap provides 0 → 5
    0 → 0px
    1 → 0.25rem (≈ 4px)
    2 → 0.5rem (≈ 8px)
    3 → 1rem (≈ 16px)
    4 → 1.5rem (≈ 24px)
    5 → 3rem (≈ 48px)
    auto → only for margins (m-auto → center alignment)
